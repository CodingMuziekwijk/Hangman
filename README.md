# Hangman
Native hangman game-app for Android

<p>Features:</p>

<ul>
  <li>homescreen</li>
    <ul>
      <li>New game</li>
    </ul>
    <ul>
      <li>Go to settings</li>
    </ul>
  <li>view highscores</li>
    <ul>
      <li>New game</li>
    </ul>
    <ul>
      <li>Go to settings</li>
    </ul>
  
  <li>adjust settings</li>
  <ul>
    <li>Mode</li>
    <ul>
      <li>Evil</li>
    </ul>
    <ul>
      <li>Normal</li>
    </ul>
    <li>Amount of lives</li>
    <ul>
      <li>"[1] .. [26]"</li>
    </ul>
    <li>Word length</li>
    <ul>
      <li>"[1] .. [26]"</li>
    </ul>
  </ul>
  
  <li>play hangman game</li>
    <ul>
      <li>Input to keyboard</li>
    </ul>
    <ul>
      <li>Go to highscore</li>
    </ul>
    <ul>
      <li>Go to settings</li>
    </ul>
</ul>
