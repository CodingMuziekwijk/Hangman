# Hangman
Native hangman app for Android

<p>Features:</p>

<ul>
  <li>start new game</li>
  <li>view highscores</li>
  <li>adjust settings</li>
  <li>play hangman game</li>
</ul>

<p>adjust settings:</p>

<ul>
  <li>Mode</li>
  <ul>
  <li>Evil</li>
  </ul>
  <li>Amount of lives</li>
  <li>Word length</li>
</ul>
